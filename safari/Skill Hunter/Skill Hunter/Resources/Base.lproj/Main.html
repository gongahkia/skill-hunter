Safari Container App

Skill Hunter

Browser extension that reformats supported Singapore legislation pages from sso.agc.gov.sg into a cleaner, reversible reading view.

Status unavailable

Open Safari’s extension settings to enable Skill Hunter for supported SSO pages.

What ships in this extension

  • Explicit user-triggered simplify toggle only.
  • No automatic page redirects, reload loops, or content changes before click.
  • No remote code, no analytics, and no off-device data transmission.
  • Scoped overlay UI that leaves the original legislation page intact underneath.

How to use it

  1. Open a supported legislation page on sso.agc.gov.sg.
  2. Switch the source site into Whole Document mode first.
  3. Click the Skill Hunter toolbar icon in Safari.
  4. Use Toggle simplified view to open or close the reading overlay.

Supported paths mirror the browser build: /Act/, /SL/, and /Bills-Supp/.

Settings

This audited release intentionally ships without user-configurable runtime settings. The extension exposes a single reversible action so the review surface stays narrow.

  • Permission model: active-tab interaction plus site access on supported SSO pages.
  • Current data use: on-device DOM parsing only.
  • Future settings must preserve the same narrow single-purpose scope.
